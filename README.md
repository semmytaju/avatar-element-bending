# 🌏 Avatar Element Bending Simulator


Master the four elements using the power of computer vision! This interactive web application allows you to control **Fire**, **Water**, **Air**, and **Earth** in real-time using your hand movements. Built with the creative coding power of **p5.js** and the precision of **MediaPipe Hands**.

<img src="/fire.png" width="400"> <img src="/water.png" width="400"> <img src="/earth.png" width="400">

## ✨ Features

- **🔥 Firebending**: Generate explosive red and orange flames that rise and flicker dynamically.
- **🌊 Waterbending**: Control fluid blue streams that wave and flow gracefully with your hand.
- **🌪️ Airbending**: Create swift, swirling gusts of wind with high-speed particle trails.
- **🪨 Earthbending**: Manipulate heavy brown and green rocks that obey gravity and burst from your palm.
- **👐 Real-Time Hand Tracking**: Your index finger acts as the bending source. Move your hand to direct the elements.
- **🎨 Dynamic Atmospheres**: The background gradient changes instantly to match the mood of each element.
- **⚡ High-Performance Graphics**: Powered by **p5.js** for smooth, artistic particle rendering at 60 FPS.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (with Dynamic Theming)
- **Creative Coding**: [p5.js](https://p5js.org/) (For particle systems and physics)
- **AI/ML**: [MediaPipe Hands](https://google.github.io/mediapipe/solutions/hands.html) by Google
- **Fonts**: Cinzel (Google Fonts) for the classic Avatar aesthetic
